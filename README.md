# Movie Browser

A premium, responsive movie browser built with **HTML5**, **CSS3**, and **Vanilla JavaScript (ES6)**. Browse, search, filter, sort, and save your favorite movies through a modern cinematic interface with smooth interactions and dynamic client-side rendering.

## Features

- Auto-rotating featured movie spotlight
- Live search by title, director, or cast
- Filter movies by genre
- Sort by rating, year, or title
- Add or remove movies from your watchlist
- Movie details modal with synopsis, runtime, director, and cast
- Quick links to trailers and streaming platforms
- Fully responsive design
- Keyboard-friendly navigation (Esc closes the modal)
- CSS-generated movie posters with custom rating badges

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## Getting Started

1. Clone the repository.

```bash
git clone https://github.com/Tayyaba-Ajmal/movie-browser.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in your browser.

No installation, build tools, or dependencies are required.
